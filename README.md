## VR Tactile Feedback
in deze game interact je met objecten in vr. Er is een basketball hoop beschikbaar waar als je de bal door de hoop krijgt je controller vibreert.

Wij heb gebruik gemaakt van:
* Oculus Rift CV1
* Oculus Touch Controllers
* Unity Engine
* Github
* Trello
